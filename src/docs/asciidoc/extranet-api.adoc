== Extranet API (숙소 파트너)

숙소 파트너가 자신의 숙소/객실을 등록하고 관리하는 API.
모든 요청에 `Authorization: Bearer {PARTNER_TOKEN}` 헤더 필요.
JWT 클레임의 `partnerId`로 서버가 `X-Partner-Id` 헤더를 자동 주입한다.

---

=== 숙소 관리

==== 숙소 등록

`POST /api/v1/extranet/accommodations`

등록 후 상태는 `PENDING`. 관리자 승인 후 `ACTIVE`로 전환된다.

===== 요청

[source,json]
----
{
  "name": "강남 그랜드 호텔",
  "type": "HOTEL",
  "full_address": "서울시 강남구 테헤란로 1",
  "latitude": 37.5012,
  "longitude": 127.0396,
  "location_description": "강남역 도보 3분",
  "check_in_time": "15:00",
  "check_out_time": "11:00",
  "supported_locales": ["ko", "en"],
  "translations": [
    {
      "locale": "en",
      "name": "Gangnam Grand Hotel",
      "full_address": "1 Teheran-ro, Gangnam-gu, Seoul",
      "location_description": "3 min walk from Gangnam Station",
      "introduction": "Luxury hotel in the heart of Gangnam",
      "service_and_facilities": "Free WiFi, Pool, Gym",
      "usage_info": "Check-in from 15:00",
      "cancellation_and_refund_policy": "Free cancellation 3 days before"
    }
  ]
}
----

[cols="1,1,1,3"]
|===
| 필드 | 타입 | 필수 | 설명

| name | String | Y | 숙소명
| type | String | Y | `HOTEL` / `MOTEL` / `PENSION` / `RESORT` / `HOSTEL` / `GUESTHOUSE` / `CAMPING`
| full_address | String | Y | 주소
| latitude | Double | Y | 위도
| longitude | Double | Y | 경도
| location_description | String | N | 위치 설명
| check_in_time | String | N | 체크인 시간 (예: "15:00")
| check_out_time | String | N | 체크아웃 시간 (예: "11:00")
| supported_locales | List<String> | N | 지원 언어 코드 목록 (`ko`, `en`, `ja`)
| translations | List<Object> | N | 다국어 번역 목록
| translations[].locale | String | Y | 언어 코드
| translations[].name | String | N | 번역된 숙소명
| translations[].full_address | String | N | 번역된 주소
| translations[].location_description | String | N | 번역된 위치 설명
| translations[].introduction | String | N | 소개글
| translations[].service_and_facilities | String | N | 서비스 및 시설
| translations[].usage_info | String | N | 이용 안내
| translations[].cancellation_and_refund_policy | String | N | 취소/환불 정책
|===

==== 숙소 수정 요청

`PUT /api/v1/extranet/accommodations/{accommodationId}`

즉시 반영되지 않고 수정 요청(ModificationRequest)이 생성된다. 관리자 승인 후 반영.

===== 요청

[source,json]
----
{
  "name": "강남 그랜드 호텔 (수정)",
  "location_description": "강남역 도보 1분",
  "translations": [
    {
      "locale": "en",
      "name": "Gangnam Grand Hotel (Updated)"
    }
  ]
}
----

===== 응답

[source,json]
----
{
  "status": "SUCCESS",
  "data": 42,
  "error": null,
  "timestamp": "2026-04-22T07:00:00Z"
}
----

`data`는 생성된 `modificationRequestId` (Long).

=== 재고 관리

==== 재고 설정

`POST /api/v1/extranet/room-options/{roomOptionId}/inventories`

날짜 범위(`start_date` ~ `end_date`) 또는 날짜 목록(`dates`)으로 일괄 설정한다.

===== 요청

[source,json]
----
{
  "start_date": "2026-05-01",
  "end_date": "2026-05-31",
  "total_quantity": 3
}
----

[cols="1,1,1,3"]
|===
| 필드 | 타입 | 필수 | 설명

| start_date | LocalDate | 조건부 | 범위 설정 시 사용 (end_date와 함께)
| end_date | LocalDate | 조건부 | 범위 설정 시 사용
| dates | List<LocalDate> | 조건부 | 특정 날짜 목록 설정 시 사용
| total_quantity | Integer | Y | 총 재고 수량 (min 0)
|===

===== 응답

[source,json]
----
{
  "status": "SUCCESS",
  "data": [
    {
      "id": 100,
      "room_option_id": 20,
      "date": "2026-05-01",
      "total_quantity": 3,
      "remaining_quantity": 3,
      "status": "AVAILABLE"
    }
  ],
  "error": null,
  "timestamp": "2026-04-22T07:00:00Z"
}
----

---

==== 재고 조회

`GET /api/v1/extranet/room-options/{roomOptionId}/inventories?start_date={date}&end_date={date}`

===== 쿼리 파라미터

[cols="1,1,1,3"]
|===
| 파라미터 | 타입 | 필수 | 설명

| start_date | LocalDate | Y | 조회 시작일
| end_date | LocalDate | Y | 조회 종료일
|===

=== 요금 관리

==== 요금 설정

`POST /api/v1/extranet/room-options/{roomOptionId}/prices`

날짜 범위 또는 날짜 목록으로 일괄 설정한다.

===== 요청

[source,json]
----
{
  "start_date": "2026-05-01",
  "end_date": "2026-05-31",
  "price_type": "STAY",
  "base_price": 150000,
  "selling_price": 130000,
  "tax_included": true
}
----

[cols="1,1,1,3"]
|===
| 필드 | 타입 | 필수 | 설명

| start_date | LocalDate | 조건부 | 범위 설정 시 사용
| end_date | LocalDate | 조건부 | 범위 설정 시 사용
| dates | List<LocalDate> | 조건부 | 특정 날짜 목록 설정 시 사용
| price_type | String | Y | `STAY` (숙박) / `HOURLY` (대실)
| base_price | BigDecimal | Y | 정가
| selling_price | BigDecimal | Y | 판매가
| tax_included | Boolean | Y | 세금 포함 여부
|===

===== 응답

[source,json]
----
{
  "status": "SUCCESS",
  "data": [
    {

      "room_option_id": 20,
      "date": "2026-05-01",
      "price_type": "STAY",
      "base_price": 150000,
      "selling_price": 130000,
      "tax_included": true
    }
  ],
  "error": null,
  "timestamp": "2026-04-22T07:00:00Z"
