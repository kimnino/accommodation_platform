== 개요

OTA 숙박 플랫폼 REST API 문서입니다. 채널별로 분리되어 있습니다.

|===
| 채널 | 대상 | Base URL

| Extranet
| 숙소 파트너
| `/api/v1/extranet`

| Admin
| 플랫폼 관리자
| `/api/v1/admin`

| Customer
| 일반 사용자
| `/api/v1`
|===

== 공통 응답 형식

모든 API는 다음 형식으로 응답합니다.

=== 성공 응답

[source,json]
----
{
  "status": "SUCCESS",
  "data": { ... },
  "error": null,
  "timestamp": "2026-04-20T06:30:00Z"
}
----

=== 에러 응답

[source,json]
----
{
  "status": "ERROR",
  "data": null,
  "error": {
    "code": "ACCOMMODATION_NOT_FOUND",
    "message": "숙소를 찾을 수 없습니다.",
    "field_errors": null
  },
  "timestamp": "2026-04-20T06:30:00Z"
}
----
